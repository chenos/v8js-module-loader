# V8Js ModuleLoader

## Requirements

- PHP 7.0+
- V8Js extension 2.0+

## Installation

```sh
composer require chenos/v8js-module-loader
```

## Testing

```
make test
```

## Example

```php
make example
```

Access http://127.0.0.1:8888

## Dependents

- [chenos/v8js-parser](https://github.com/chenos/v8js-parser)
